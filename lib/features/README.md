# features

機能単位の構成。

各 feature は基本的に以下で分ける。

- `data`: API、Firestore、DTO、repository 実装
- `domain`: entity、repository 契約、use case
- `presentation`: controller、binding、screen、widget

主な機能領域:

- `auth`
- `calendar`
- `meeting`
- `client`
- `business_card`
- `notification`
- `location_trigger`
- `spreadsheet_sync`
- `telephony`
- `settings`
